# 事件总线
创建一个新的vue对象作为eventBus,然后该对象使用$on和$emit
$on(string/Array,funciton)
参数一是要监听的事件名
参数二是回调函数
$emit(eventName,[...args])
触发当前实例上的事件，附加参数会传递给监听器回调函数。

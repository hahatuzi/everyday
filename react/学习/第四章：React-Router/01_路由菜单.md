# 一：路由模式
  ### 1.模式一：HashRouter  会通过url地址中的hash值来对地址进行匹配
  ### 2.模式二：BrowserRouter  直接通过url地址进行组件的跳转，使用过程和url没有区别

# 一：路由分级
  ```js
    <BrowserRouter>
    {/* 导航区 */}
      <Link to="/login"></Link>
      {/* 展示区 */}
      <Routes>
        <Route path='/home' element={<Home />} />
        <Route path='/user-manage/list' element={<UserList />} />
        <Route path='/right-manage/role/list' element={<RoleList />} />
        <Route path='/right-manage/right/list' element={<RightList />} />
      </Routes>
    </BrowserRouter>
  ```

# 二：动态路由
  ```js
    <Route path="student/:id" element={<Student></Student>}></Route>
    <Route path="student/:id/detail" element={<Student></Student>}></Route>
  ```

# 三：路由跳转
  ### 1.路由跳转方式
  - (1)声明式导航跳转:Link,NavLink
  - (2)useNavigate()
    ```js
      <Link  to="/home"></Link>
      <navLink to=""></navLink>
      // useNavigate()
      import  {useNavigate}  from 'react-route-dom'
      const  navigate = useNavigate()
      navigate('/home',{replace: true}) //  replace是否删除历史记录
      navigate(1)
      navigate(-1)
      navigate(-2)
    ```
  ### 1.重定向：
    ```js
      // index属性来指定默认路由
      // replace属性用于控制跳转模式（push 或 replace，默认是push）
      <Route index element={<Navigate to='/home' />} />
      //  404  path="*"不能省略
      <Route path="*" element={ <NotFound />} />
    ```

# 四：路由参数传递以及获取
  ### 1.Link组件方式
  - (1)动态路由传参
    ```js
      <Link excat to='/detail/1'>
      <Link excat to='/detail/2'>
      // 通过props.match.params获取参数
      console.log(props.match.params)
    ```
  - (2)查询参数传递:路径拼接参数
    ```js
      <Link to='/demo/test?name=tom&age=18'}>
      console.log(props.location.search)
    ```
  - (3)state参数方式：
    ```js
      <Link  to={ { pathname: '/login',  state: { name: 'lisa' } } }
      console.log(props.location.state)
    ```
  ### 2.navigate()方式
  - (1)useSearchParams()
  - (2)useLocation()
  - (3)useParams()
    ```js
      import  {useNavigate}  from 'react-route-dom'
      const  navigate = useNavigate()
      // 路由参数携带方式包括三种：search，params,state
      navigate('/page1?name=jack&age=18')  // search----通过useSearchParams()来获取search数据
      const [searchParams] = useSearchParams()
      console.log(searchParams.get('name'))
      navigate('/page1/kack')     // params----通过useParams()来获取params数据
      navigate('/page1', {state:{data:'test'})  // state----通过useLocation()来获取state数据,获取地址信息
      console.log(useLocation().state)
      useMatch('/about')// 检查当前url是否匹配某个路由
    ```

2.运算符：+，-，*，/，%，==
  (1)任何类型加String均为String
  (2)NAN:not a number,Number类型,当运算出现错误时结果会变成NAN.如：a = 0/0
  (3)NAN与包括自己在内的任何东西都不相等。
3.循环：
for循环和while循环
```js
// for (var i = 0;i<10;i++){}其实可以转换变形为以下形式:
//但中间的判断条件一定要注意，因为i如果初始值为0的话，判断会默认转化为false
var i = 1
for(;i<10;){
  i++
  console.log(i)
}
````

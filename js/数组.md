# 数组常见的删除元素的七大方法：
1.通过length来删除元素，如：
```js
let arr = [1,2,3] //length=3
arr.length = 2 // 这样的话就可以取巧删掉最后一个元素。
```
2.delete关键字
如：delete arr[1]，但是delete只是将该位置饿元素指向变成空，数组长度其实不变。
3.栈方法
pop()
4.队列方法：
shift() //先进先出原则，所以删除的是第一个元素。
5.操作方法：splice()
6.迭代。循环或者filter
```js
var arr = [1,2,3,4]
arr.forEach()
arr.filter(item=>{
  return item !== 2
})
// [1,3,4]
```
7.向数组原型上添加方法来达到删除的目的

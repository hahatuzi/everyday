# 希尔排序：
是插入排序的一种，又称为缩小增量排序。
**原理**
选定一个增长量h,将增长量h作为数据分组的依据。
对分组好的每一组数据完成插入排序
减小增长量，最小增长量为1，重复完成插入排序。
增长量h的确定规则
```js
while(h<length/2){
  h = 2h+1
}
// h的减小规则：
h = h/2
```

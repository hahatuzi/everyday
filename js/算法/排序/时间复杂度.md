  ### 3.算法模型对比结论：
  （1）算法函数中的常熟可以忽略

  （3）算法函数中最高次幂越小，算法效率越高
# 大O表示法
只要高阶项，不要低阶项，也不要高阶项的系数
  记法规则：
  （1）用1代替所有加法常数
  （2）运行次数仅保留最高阶项
  （3）如果最高阶项存在，且 常数因子不为1，我们去除这个项的常数
  比如：(2n^3+3) --> (n^3)
# 函数调用的时间复杂度
  ```js
  function fn () {
    int n =100
    for(var i = 0, i<n; i++){ 
      this.show() //执行1次
    }
  }
  function show () {
    console.log('ok)
  }
  fn()
  // 时间复杂度为O(n)
  ```
  ```js
  function fn () {
    // for循环自身执行n次
    for(var i = 0, i<n; i++){
      this.show()
    }
  }
  function show () {
    // 自身执行n次
    for(var i = 0, i<n; i++){
      sum +=i
    }
  }
  fn()
  // 时间复杂度为O(n^2)
  ```

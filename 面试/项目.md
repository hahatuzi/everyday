# 2.函数式编程的理解，怎么理解函数是一等公民
  - 函数可以作为变量，回调，函数参数
  - 纯函数：不依赖外部变量
    ```js
      // 纯函数
      function sum (a, b) {
        return a + b
      }
      //非纯函数
      let count = 2
      function sum (a, b) {
        return a + b + count // 依赖了外界的count变量
      }
    ```
  - 函数柯里化

## 二、阶段一：Java 基础语法

> 周期：4-6 周 | 目标：能写控制台程序，掌握编程基本概念
> 建议：**每天至少写 150 行代码**

### 2.1 环境搭建（1 天）

| 任务 | 说明 |
|------|------|
| 安装 JDK 17 | Oracle 官网或 OpenJDK 下载，配置 JAVA_HOME |
| 安装 IntelliJ IDEA | 社区版免费，够用 |
| Hello World | 写出第一个 Java 程序 |

```java
// 你的第一行 Java 代码
public class HelloWorld {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

### 2.2 基础语法（2 周）

| 知识点 | 学习重点 | 练习量 |
|--------|---------|--------|
| 变量与数据类型 | 8 种基本类型、类型转换、String | 每天 20 题 |
| 运算符 | 算术、关系、逻辑、三元、位运算 | 10 题 |
| 流程控制 | if-else、switch、for、while、do-while | 30 题 |
| 数组 | 声明、初始化、遍历、排序、二维数组 | 15 题 |
| 方法 | 定义、调用、重载、递归 | 10 题 |

```java
// 阶段练习：ATM 取款机
```

**本阶段检验标准：**

## 三、阶段二：Java 面向对象与核心类库

> 周期：4-6 周 | 目标：理解面向对象思想，掌握 Java 常用 API

## 四、阶段三：数据库与 JDBC

> 周期：3-4 周 | 目标：能设计简单数据库，用 JDBC 操作数据

### 4.1 MySQL 基础（2 周）

| 知识点 | 要求 |
|--------|------|
| 安装与连接 | MySQL 8.0 安装、Navicat/DBeaver 客户端 |
| DDL | CREATE/DROP/ALTER 数据库和表 |
| DML | INSERT/UPDATE/DELETE |
| DQL | SELECT、WHERE、ORDER BY、LIMIT |
| 聚合函数 | COUNT/SUM/AVG/MAX/MIN、GROUP BY、HAVING |
| 多表查询 | 内连接、左外连接、子查询 |
| 约束 | 主键、外键、唯一、非空、默认值 |
| 事务 | 开启/提交/回滚、ACID 概念 |

```sql
-- 核心练习 SQL
-- 1. 单表查询
SELECT name, salary FROM employee 
WHERE dept_id = 1 AND salary > 5000 
ORDER BY salary DESC LIMIT 10;

-- 2. 多表连接（必会！）
SELECT e.name, d.dept_name, e.salary
FROM employee e
LEFT JOIN department d ON e.dept_id = d.id;

-- 3. 分组统计
SELECT dept_id, AVG(salary) avg_sal, COUNT(*) cnt
FROM employee GROUP BY dept_id HAVING cnt > 3;
```

### 4.2 JDBC（1 周）

| 知识点 | 要求 |
|--------|------|
| 注册驱动 | Class.forName / DriverManager |
| 连接数据库 | Connection、URL 格式 |
| Statement | 基本执行（有 SQL 注入风险，了解即可） |
| **PreparedStatement** | 预编译、防注入（重点掌握） |
| ResultSet | 结果集遍历、字段获取 |
| JDBC 工具类 | 封装连接/释放，抽取 DBUtils |
| Druid 连接池 | 配置、使用（了解连接池概念） |

```java
// JDBC 标准操作步骤（面试常问）
// 1. 注册驱动
// 2. 获取连接
// 3. 创建 PreparedStatement
// 4. 设置参数
// 5. 执行 SQL
// 6. 处理结果集
// 7. 释放资源（逆序释放）
```

### 7.3 项目规范（从一开始就养成好习惯）

```java
// 标准分层结构
com.example.project
├── controller    // 控制器层：接收请求、参数校验
├── service       // 服务层：业务逻辑
│   └── impl
├── mapper        // 持久层：数据库操作（MyBatis 接口）
├── entity        // 实体类：对应数据库表
├── dto           // 数据传输对象
├── vo            // 视图对象（返回给前端）
├── common        // 公共类：Result 封装、异常处理、工具类
└── config        // 配置类：拦截器、跨域等
```

## 学习节奏建议

```
每日投入 2-4 小时，总周期约 4-6 个月

基础阶段（前 3 个月）：
  ├── 60% 时间：看视频 + 敲代码（一定要动手！）
  ├── 20% 时间：做练习题（每节后的练习必须做）
  └── 20% 时间：整理笔记（用自己的话复述）

框架阶段（第 4-5 个月）：
  ├── 40% 时间：看视频 + 跟着敲
  ├── 40% 时间：自己写 Demo（脱离视频！）
  └── 20% 时间：总结 + 面试题

项目阶段（第 5-6 个月）：
  ├── 70% 时间：独立编码（遇到问题优先自己解决）
  └── 30% 时间：复盘优化 + 写文档
```

### 避坑指南（过来人的经验）

1. **不要只看不写**：看视频 10 遍不如自己敲 1 遍
2. **不要死磕细节**：第一遍看不懂的先跳过，后面自然就懂了
3. **不要跳过面向对象**：这是 Java 的灵魂，必须彻底理解
4. **不要忽视数据库**：面试 70% 的 SQL 题都来自基础阶段
5. **不要一上来就学微服务**：单体项目都没写过，微服务等于空中楼阁
6. **不要只看一套教程**：一个知识点看不懂就换一个老师的讲法

## 阶段完成自检清单

### 基础阶段 ✅
- [ ] 能独立写出冒泡排序、二分查找
- [ ] 能说出封装、继承、多态的含义和代码体现
- [ ] 理解 String 的不可变性
- [ ] 会用 ArrayList 和 HashMap
- [ ] 能处理 try-catch 异常

### 数据库阶段 ✅
- [ ] 能建表、写增删改查 SQL
- [ ] 会写内连接和左连接查询
- [ ] 理解事务的基本概念
- [ ] 能用 JDBC 连接数据库操作数据

### Web 阶段 ✅
- [ ] 理解 HTTP 请求/响应格式
- [ ] 能说出 GET 和 POST 的区别
- [ ] 理解 Servlet 生命周期
- [ ] 能说出转发和重定向的区别
- [ ] 理解 Cookie 和 Session 的作用

### 框架阶段 ✅
- [ ] 能用 Maven 管理依赖
- [ ] 能用 MyBatis 写动态 SQL
- [ ] 理解 Spring IoC 和依赖注入
- [ ] 能用 Spring Boot 快速搭建项目
- [ ] 能写出 RESTful 风格的 CRUD 接口

### 项目阶段 ✅
- [ ] 独立完成一个完整项目
- [ ] 代码分层清晰（Controller → Service → Mapper）
- [ ] 有统一的异常处理和返回格式
- [ ] 能用 Git 管理代码

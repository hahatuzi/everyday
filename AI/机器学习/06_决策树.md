# 决策树：
  决策树是一种树形结构，每个**节点**都是特征的判断，**叶子结点**表示分类结果
  ## 流程：
  - 特征选择
  - 决策树生成
  - 决策树处理
  > 优先选择信息增益大的特征作为节点
# 信息熵：( - 标签列A的占比 * log2(标签列的占比))+ ( - 标签列B的占比 * log2(标签列B的占比))
  > 信息熵用标签列，特征熵用特征列
# 条件熵：特征A的占比 * 特征A的信息熵 + 特征B的占比 * 特征B的信息熵
# 信息增益：信息熵 - 条件熵
# 特征熵：( - 特征A的占比 * log2(特征A的占比))+ ( - 特征B的占比 * log2(特征B的占比))
# 信息增益率 = 信息增益/特征熵
  > 信息增益率偏向于选择种类多的特征作为分裂依据

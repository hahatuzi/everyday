## 一、大模型原理
  ### 1.1 核心本质
  - (1)词嵌入：将文字转化为向量:将文字转化为token,为每个token生成包含语义的向量，模型通过计算向量的距离来理解语义的关联性
  - (2)注意力机制：生成内容的时候忽略无关词汇，能够准确判断句子中各成分的核心关联度。多个注意力头分别捕获语义，逻辑等不同维度的信息
  ### 1.2 五个核心阶段
  - 高质量数据准备与治理
  - 预训练阶段
  - 微调阶段
  - 对齐与强化学习
  - 部署与迭代优化
  ### 1.3 监督微调
  > 在预训练模型基础上，使用高质量数据集进行小范围参数调整，使模型学会理解并遵守指令
  - 监督：带着参考答案训练数据。

## 二、AI Agent
  ### 2.1 什么叫AI Agent
  > Agent = **LLM** (大脑) + **Planning** (规划) + **Tool use** (执行) + **Memory** (记忆)。
  - LLM (大脑)： 作为核心推理机，负责理解意图、生成文本和进行逻辑判断。
  - 规划Planning： 能够将复杂的目标（如"帮我策划一场技术沙龙"）拆解成可执行的步骤，最重要的是让LLM具备**子任务拆解**和**反思**能力，让它形成思维树和思维链。
  - 记忆Memory： 包括短期和长期两种。短期记忆：执行上下文。长期记忆一般指外部知识库。
  - 工具使用Tool Use： 能够根据需求去查谷歌搜索、读数据库、甚至跑 Python 代码。
---
  ### 2.2 AI Agent增强的方法
  > 普通的 LLM 只是 One-shot（一次性） 的响应，而 Agent 的核心在于 Iterative（迭代）。ReAct 模式 (Reason + Act) 是目前最主流的 Agent 推理逻辑，它通过**推理**和**行动**来增强推理和决策的效果
  - 1.Thought (思考)： 模型描述当前要做什么，为什么要这么做。
  - 2.Action (行动)： 模型选择一个工具（如：Google Search）。
  - 3.Observation (观察)： 模型读取工具返回的结果。
  - 4.Repeat (循环)： 重复上述步骤，直到得出最终答案。

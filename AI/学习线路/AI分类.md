## 完整分类（按技术栈从深到浅）

### 第一层：算法研究类（造轮子）

| 岗位 | 做什么 | 门槛 | 代表公司 |
|------|--------|------|---------|
| **AI 研究科学家** | 发论文、发明新算法/新架构 | 🔴 极高（PhD为主） | OpenAI、DeepMind、FAIR |
| **NLP 研究员** | 研究语言模型新方法 | 🔴 极高 | 各大厂 AI Lab |
| **CV 研究员** | 研究图像/视频新算法 | 🔴 极高 | 各大厂 AI Lab |
| **RL 研究员** | 强化学习算法研究 | 🔴 极高 | DeepMind、OpenAI |

**需要**：顶会论文、数学功底、PyTorch/JAX 精通

---

### 第二层：模型训练类（造发动机）

| 岗位 | 做什么 | 门槛 | 代表公司 |

| **预训练工程师** | 从零训练基座模型 | 🔴 高 | 各大模型公司 |

## 三者的关系

```
人工智能 (AI)
  │
  ├── 机器学习 (ML)           ← ML 工程师在这里
  │     │
  │     ├── 传统ML（XGBoost、随机森林...）
  │     │
  │     └── 深度学习 (DL)     ← DL 工程师在这里
  │           │
  │           └── 大语言模型 (LLM)
  │                 │
  │                 └── AI Agent  ← Agent 工程师在这里
  │
  └── 其他AI（规则系统、搜索算法...）
```

**包含关系**：DL 是 ML 的一个分支，LLM 是 DL 的一个产物，Agent 是 LLM 的一种应用方式。

## ML 工程师 vs DL 工程师 vs Agent 工程师

| 维度 | ML 工程师 | DL 工程师 | Agent 工程师 |
|------|----------|----------|-------------|
| **核心技能** | 特征工程 + 传统算法 | 网络设计 + 训练调优 | Prompt + 工具编排 |
| **是否需要训练模型** | ✅ 是 | ✅ 是 | ❌ 不需要 |
| **是否需要 GPU** | 不一定 | ✅ 必须 | ❌ 不需要 |
| **数学要求** | 中等（概率统计） | 高（线性代数+微积分） | 低（基本概念即可） |
| **入门难度** | 中等 | 高 | 低 |
| **典型岗位** | 推荐/风控/搜索 | CV/NLP/语音 | AI应用/智能体开发 |

## 入门建议

如果你现在想快速入行 AI 领域，**AI Agent 工程师**是最佳切入点：

- ✅ 不需要数学博士背景
- ✅ 不需要会训练模型
- ✅ 会 Python + 懂 API 就能开始
- ✅ 市场需求正在爆发
- ✅ 未来可横向扩展：RAG 工程师 → AI 应用架构师 → LLM 应用架构师

**Agent 工程师本质上是应用层工程师**，不是算法工程师。模型能力由 OpenAI/DeepSeek/Anthropic 等公司提供，Agent 工程师负责把模型能力编排成可用的产品。

```
  → 调参 → 评估 → 部署模型                  → 搭建工作流 → 工程化落地
        │                                         │
        ▼                                         ▼
    自己"造"模型                              用现成的模型"搭"应用
```

## Agent 工程师需要学 ML/DL 吗？

| 阶段 | 需要程度 | 具体内容 |
|------|---------|---------|
| **初级 Agent** | ❌ 不需要 | 零 ML/DL 基础也能入门 |
| **中级 Agent** | ⚠️ 少量概念 | 理解 Token/Embedding/向量相似度即可 |
| **高级 Agent** | ✅ 需要了解 | Transformer 原理、RAG 优化、评估方法论 |

大多数 Agent 岗位走纯应用层路线，把时间花在 **API 调用、Prompt 工程、Tool Calling、RAG 和工作流编排**上，比啃机器学习课程 ROI 高得多。

## 学习建议

1. **不要贪多**：每个岗位选 1-2 个课程深入学，比收藏 20 个课有用
2. **边学边做**：看完一节立刻敲代码/实操，不要只看不练
3. **交叉学习**：Agent + RAG + Prompt 三者的课可以并行看，知识互补
4. **关注更新**：AI 领域迭代快，优先看 2025-2026 年的新课
5. **GitHub 配合**：很多课程配套 GitHub 源码，clone 下来跑一遍效果翻倍

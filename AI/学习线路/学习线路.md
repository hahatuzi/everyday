# 第二阶段：前端AI开发（1.5-2个月）
  ### AI API 调用
  - 主流API：
    - OpenAI API (GPT-4/GPT-4o)
    - Claude API (Anthropic)
    - Google Gemini API
    - 国产：文心一言、通义千问、智谱GLM
  ### JavaScript AI SDK
    - OpenAI Node.js SDK
    - LangChain.js: 强大的AI应用开发框架
    - Vercel AI SDK: 专为前端设计的AI SDK
    - @ai-sdk/openai: 支持多模型
  ### 必备概念
    - 流式响应（Streaming）
    - 函数调用（Function Calling）
    - Token计算
    - 系统提示词设计
  ### 实践技能
    ```js
      // 基础调用示例
      import { openai } from '@ai-sdk/openai';
      import { generateText } from 'ai';

      const result = await generateText({
        model: openai('gpt-4-turbo'),
        prompt: '用户问题',
        system: '你是一个专业助手',
      });
    ```
# 第三阶段：AI前端框架与工具（1-2个月）
  ### 低代码/AI工作流
    - Dify: 可视化AI应用编排
    - Coze: 字节跳动AI应用平台

    - 文本处理:natural,compromise

    - 语音识别:Whisper.js
    - 代码辅助:GitHub Copilot API
# 第四阶段：实战项目与进阶（持续学习）
  ### 初级项目
    - AI对话助手: 对话历史、流式输出、多模型切换
    - AI写作助手: 文章续写、语法纠错、翻译
    - 智能客服机器人: RAG知识库问答
  ### 中级项目
    - AI数据分析看板: 自然语言查询数据库
    - AI图片生成界面: 调用SD/Midjourney API
    - 代码生成器: 根据描述生成代码片段
    - PDF文档智能分析: 提取摘要、回答问题
  ### 高级项目
    - 企业级AI知识库: 向量数据库 + RAG + 多模态
    - AI Agent系统: 自主任务执行、工具调用
    - 实时语音对话应用: 语音识别 + LLM + 语音合成
  ### 技术栈组合推荐
    - 前端: Next.js/React + TypeScript + Tailwind CSS
    - AI层: LangChain.js + Vercel AI SDK
    - 后端: Next.js API Routes / Express
    - 数据库: PostgreSQL + Pinecone/Weaviate (向量)
    - 部署: Vercel
